# 👟 Adidas US Sales Analysis - Power BI Dashboard

## 📌 Project Overview
This project is a Data Analysis and Visualization dashboard built using **Microsoft Power BI**. It tracks and analyzes the sales performance of Adidas across various regions, retailers, and product categories in the United States.

## 📂 Files Included in this Repository
1. **`Adidas_sales.pbix.zip`**: This compressed file contains the main Power BI Dashboard (`.pbix`). It includes all the visual reports, charts, and data models. *(Note: Please extract this ZIP file to access the dashboard).*
2. **`Adidas US Sales Datasets.xlsx`**: The raw dataset used for this analysis. It contains detailed sales records, including Retailer, Region, Product, Units Sold, Total Sales, and Operating Margin.

## 🚀 How to Run and View This Project
Follow these steps to explore the dashboard on your local machine:

**Step 1: Download the Repository**
* Click the green **`Code`** button at the top right of this repository and select **`Download ZIP`**.

**Step 2: Extract the Dashboard File**
* Extract the downloaded repository folder. 
* Navigate inside the folder, locate **`Adidas_sales.pbix.zip`**, and extract/unzip it to get the actual `Adidas_sales.pbix` file.

**Step 3: Install Power BI Desktop**
* You need **Microsoft Power BI Desktop** installed on your system to view and interact with this file. You can download it for free from the [official Microsoft website](https://powerbi.microsoft.com/desktop/).

**Step 4: Open the Dashboard**
* Double-click the extracted **`.pbix`** file.
* The dashboard will open in Power BI, allowing you to interact with the slicers and visual insights!

## 💡 Key Insights & Tech Stack
* **Tool Used:** Microsoft Power BI
* **Data Source:** Excel Dataset (`.xlsx`)
* **Data Processing:** Power Query (Data cleaning and formatting)
* **Features:** Sales trend analysis, geographical performance tracking, and product profitability insights.
